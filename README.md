# Adwaita Wallpaper

Лёгкое приложение для Windows, которое **автоматически меняет обои рабочего стола при переключении светлой/тёмной темы Windows**. Оформление — в стиле GNOME Adwaita (как в свежей Fedora).

## Возможности

- 🌗 Отдельные обои для **дневной** (светлой) и **ночной** (тёмной) темы
- ⚡ **Автосмена** обоев в момент, когда Windows переходит в светлый/тёмный режим
- 🖼 Выбор изображений и режим расположения (заполнение / вписать / растянуть / по центру / замостить / на все мониторы)
- 📌 Живёт в **системном трее**, продолжает работать после закрытия окна
- 🚀 **Автозапуск** при входе в Windows (стартует свёрнутым в трей)
- 🎨 Красивый интерфейс Adwaita со светлой и тёмной палитрой

## Как работает

- Тема Windows отслеживается через `nativeTheme` Electron (реагирует на изменение
  «Настройки → Персонализация → Цвета → Выбор режима»).
- Обои ставятся напрямую через Windows API (`SystemParametersInfo`, вызываемый из
  PowerShell P/Invoke) — без сторонних бинарников.
- Настройки хранятся в `%APPDATA%\adwaita-wallpaper\config.json`.

## Запуск из исходников

```powershell
npm install
npm start
```

## Сборка установщика

```powershell
npm run dist
```

Готовый установщик появится в папке `dist/` (`Adwaita Wallpaper Setup x.x.x.exe`).

## Структура

| Файл | Назначение |
|------|-----------|
| `main.js` | Главный процесс: трей, отслеживание темы, смена обоев, автозапуск, IPC |
| `preload.js` | Безопасный мост `window.api` между интерфейсом и системой |
| `renderer/` | Интерфейс (HTML/CSS/JS) в стиле Adwaita |
| `assets/` | Иконки приложения и трея |

## Требования

Только **Node.js** для сборки. Никаких Visual Studio, JDK или Android SDK не нужно.
