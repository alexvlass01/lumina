# Lumina

**Lumina** автоматически меняет обои рабочего стола Windows при переключении светлой и тёмной
темы — у каждого монитора своя пара обоев. Лёгкое приложение в стиле GNOME Adwaita: живёт в
системном трее и работает в фоне.

## Возможности

- 🌗 **Своя пара обоев для дня и ночи.** Включил тёмный режим Windows — встали ночные обои, вернул светлый — дневные.
- 🖥 **Разные обои на каждый монитор.** Карта мониторов в реальных пропорциях; для каждого экрана задаётся отдельная пара день/ночь.
- ⚡ **Автосмена** в момент переключения темы Windows — без ручных действий.
- 🎚 **Режимы расположения:** заполнение, вписать, растянуть, по центру, замостить, на все мониторы.
- 📌 **Живёт в трее**, продолжает работать после закрытия окна.
- 🚀 **Автозапуск** при входе в Windows (стартует свёрнутым в трей).
- 🌍 **Три языка интерфейса** — українська, русский, English; режим «по системе» (язык Windows).
- 🎨 **Красивый интерфейс** Adwaita со светлой и тёмной палитрой и нативной строкой заголовка.

## Как это работает

- Тема Windows отслеживается через `nativeTheme` Electron (реагирует на «Параметры →
  Персонализация → Цвета → Выбор режима»).
- Обои на каждый монитор ставятся через системный COM-интерфейс Windows
  `IDesktopWallpaper` — без сторонних бинарников.
- Выбранные изображения копируются в данные приложения, поэтому не «слетают» при обновлении
  или перемещении оригинала.

## Установка и запуск

Готовое приложение — портативное, ставить ничего не нужно: запусти
`Lumina.exe` из папки сборки.

### Из исходников

```powershell
npm install
npm start
```

### Сборка портативного приложения

```powershell
node node_modules/@electron/packager/bin/electron-packager.mjs . "Lumina" `
  --platform=win32 --arch=x64 --icon=assets/icon.ico --out=dist --overwrite --app-version=1.0.1
```

Результат — `dist/Lumina-win32-x64/Lumina.exe`.

## Стек

Electron + чистый HTML/CSS/JS, без тяжёлых фреймворков и тулчейнов — ставится одним
`npm install` поверх Node. Только Windows.
