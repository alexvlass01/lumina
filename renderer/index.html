Lumina

Добро пожаловать в Lumina

Lumina автоматически меняет обои рабочего стола при переключении светлой и тёмной темы Windows. Приложение живёт в трее и работает в фоне.

Язык
Автосмена обоев
Менять обои при переключении темы Windows
Автозапуск с Windows
Запускать в фоне (в трее) при входе в систему
Автопереключение темы Windows
Lumina сама будет менять тему по расписанию. Подробнее — в «Оформлении».
Ярлык на рабочем столе
Ярлык в меню «Пуск»

Окно можно закрыть — приложение продолжит работать в трее. Повторно запускать установщик не нужно.
